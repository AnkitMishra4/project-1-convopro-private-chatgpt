# ConvoPro Private ChatGPT

## 📖 Overview

ConvoPro Private ChatGPT is a private AI chatbot application built with Python. It enables users to interact with an AI model in a secure environment while supporting conversational workflows.

## 🚀 Features

- AI-powered conversational interface
- Private chat sessions
- Clean and user-friendly interface
- Fast response generation
- Easy project setup
- Modular project structure

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/<your-username>/<repository-name>.git
```

2. Navigate to the project

```bash
cd project-1-convopro-private-chatgpt
```

3. Create a virtual environment

```bash
py -3.13 -m venv .venv
```

4. Activate the virtual environment

Windows:

```powershell
.\.venv\Scripts\Activate.ps1
```

5. Install dependencies

```bash
pip install -r requirements.txt
```

## ▶️ Running the Project

```bash
python main.py
```

*(Replace `main.py` with your project's entry point if it's different.)*
